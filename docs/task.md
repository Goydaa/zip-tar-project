# Техническое задание 
 
## 1. Описание задачи 
Разработать скрипт для работы с ZIP и TAR.GZ архивами. 
 
## 2. Цель работы 
Научиться программно создавать и распаковывать архивы. 
 
## 3. Функциональные требования 
1. Создание ZIP-архива с рекурсивным обходом папок 
2. Создание TAR.GZ архива 
3. Вывод списка файлов в архиве 
4. Распаковка архива с сохранением структуры 
5. Обработка ошибок 
 
## 4. Нефункциональные требования 
- Python 3.8+ 
- Только стандартная библиотека 
- Документированный код 
 
## 5. Ограничения 
- ZIP больше 4ГБ требует allowZip64 
- TAR.GZ не поддерживает добавление файлов после создания 
 
## 6. Источник ТЗ 
Самостоятельная разработка
