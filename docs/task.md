## 1. ОПИСАНИЕ ЗАДАЧИ 
 
**Что мы делаем?** 
 
Мы разрабатываем скрипт для программной работы с архивами. 
Скрипт должен уметь: 
1. Создавать ZIP-архивы из любой папки 
2. Создавать TAR.GZ архивы 
3. Показывать содержимое архива без распаковки 
4. Распаковывать архивы 
 
**Зачем?** 
- Автоматически создавать резервные копии 
- Упаковывать файлы для отправки 
- Распаковывать скачанные дистрибутивы

## 3. ФУНКЦИОНАЛЬНЫЕ ТРЕБОВАНИЯ 
 
### 3.1. Создание ZIP-архива 
- Функция `create_zip(source_dir, output_path)` 
- Рекурсивно обходит все вложенные папки 
- Сохраняет структуру директорий 
- Использует сжатие DEFLATED 
 
### 3.2. Создание TAR.GZ архива 
- Функция `create_targz(source_dir, output_path)` 
- Сжатие GZIP 
- Сохраняет структуру папок 
 
### 3.3. Просмотр содержимого 
- Функция `list_archive(archive_path)` 
- Определяет тип архива (ZIP или TAR) 
- Выводит список файлов с размерами 
 
### 3.4. Распаковка 
- Функция `extract_archive(archive_path, extract_dir)` 
- Создаёт целевую папку, если её нет 
- Сохраняет структуру папок 
 
### 3.5. Интерфейс командной строки 
- `--source` - указать папку для архивации 
- `--extract` - распаковать архив 
- `--list` - показать содержимое 
- `--zip-output` - имя ZIP-архива 
- `--tar-output` - имя TAR.GZ архива

## 5. ОГРАНИЧЕНИЯ 
 
- ZIP-архивы больше 4ГБ требуют флага `allowZip64` 
- TAR.GZ не поддерживает добавление файлов после создания 
- Символические ссылки обрабатываются как обычные файлы
