# Выводы по работе 
 
## 1. Что было изучено 
- Создание ZIP и TAR.GZ архивов 
- Рекурсивный обход директорий через os.walk 
- Чтение архива без распаковки 
- Распаковка с сохранением структуры 
 
## 2. С какими трудностями столкнулся 
- Пустые директории не добавляются в ZIP автоматически 
- Определение типа архива через is_zipfile и is_tarfile 
 
## 3. Что получилось удачно 
- Поддержка обоих форматов 
- Понятные сообщения об ошибках 
 
## 4. Где инструмент уместен 
- Резервное копирование 
- Экспорт файлов 
- Упаковка логов 
 
## 5. Где инструмент неуместен 
- Когда нужна максимальная скорость 
- Для очень больших архивов >10ГБ
