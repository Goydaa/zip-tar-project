## 2. С КАКИМИ ТРУДНОСТЯМИ СТОЛКНУЛСЯ 
 
### 2.1. Технические проблемы 
 
**Проблема 1: Пустые папки не добавляются в ZIP** 
Решение: Пришлось явно добавлять папки с завершающим слешем 
 
**Проблема 2: Относительные пути** 
Решение: Использовал `os.path.relpath()` для правильного сохранения структуры 
 
**Проблема 3: Кодировка в консоли** 
Решение: Команда `chcp 65001` переключает на UTF-8
