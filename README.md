# ZIPFILE и TARFILE - Работа с архивами в Python 
 
## 1. Краткое описание инструмента 
zipfile и tarfile - это встроенные модули Python для работы с ZIP и TAR архивами. 
 
## 2. Цель изучения 
Научиться программно работать с архивами: создавать, распаковывать, читать содержимое. 
 
## 3. Описание реализованной задачи 
Создан скрипт, который создает ZIP и TAR.GZ архивы, выводит содержимое и распаковывает. 
 
## 4. Инструкция по запуску 
1. Перейдите в папку src 
2. Запустите: python archive_manager.py --source ../test_data 
 
## 5. Общие выводы 
Модули удобны, встроены в Python, не требуют установки.
