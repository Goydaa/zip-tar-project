## 📖 ЧТО ТАКОЕ ZIPFILE И TARFILE? 
 
**zipfile** и **tarfile** - это встроенные модули Python для работы с архивами. 
Представьте, что у вас есть 100 файлов, которые нужно отправить по почте. 
Вместо того чтобы прикреплять 100 файлов по отдельности, вы упаковываете их в один ZIP-архив. 
Эти модули позволяют делать это программно из Python-кода.

## 🏗️ КАК ЭТО РАБОТАЕТ? 
 
**ZIP** - самый популярный формат архивов. Поддерживается везде: Windows, Mac, Linux, телефоны. 
 
**TAR.GZ** - формат, популярный в Linux. Сначала файлы упаковываются в TAR (без сжатия), 
потом сжимаются GZIP. Обычно используется для дистрибутивов программ.

## 🔧 РАЗБОР КОДА 
 
### Импорт модулей 
```python 
import os           # для работы с файлами и папками 
import zipfile      # для ZIP-архивов 
import tarfile      # для TAR-архивов 
import argparse     # для обработки аргументов командной строки 
``` 
 
### Создание ZIP-архива 
```python 
def create_zip(source_dir, output_path): 
    with zipfile.ZipFile(output_path, 'w', zipfile.ZIP_DEFLATED) as zipf: 
        for root, dirs, files in os.walk(source_dir): 
            for file_name in files: 
                file_path = os.path.join(root, file_name) 
                arcname = os.path.relpath(file_path, os.path.dirname(source_dir)) 
                zipf.write(file_path, arcname) 
``` 
 
**Объяснение:** 
- `ZipFile(output_path, 'w', ZIP_DEFLATED)` - открываем архив на запись со сжатием 
- `os.walk(source_dir)` - рекурсивно обходим все папки и файлы 
- `zipf.write(file_path, arcname)` - добавляем файл в архив

## ❓ ЧАСТЫЕ ВОПРОСЫ 
 
**В: В чем разница между ZIP и TAR.GZ?** 
О: ZIP сжимает каждый файл отдельно. TAR.GZ сначала объединяет все файлы в один TAR-архив, 
   потом сжимает его целиком. TAR.GZ лучше сжимает много маленьких файлов. 
 
**В: Как добавить файл в существующий ZIP?** 
О: Используйте режим `'a'`: `with zipfile.ZipFile('archive.zip', 'a') as zipf:` 
 
**В: Почему не добавляются пустые папки?** 
О: ZIP не хранит пустые папки по умолчанию. Чтобы добавить, нужно написать `zipf.write(folder_path, folder_path + '/')`
